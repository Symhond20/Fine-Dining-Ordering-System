# Fine Dining Ordering System 
A Python-Based program application where  designed to manage the process of handling orders, reservations, and payments. It provides user-friendly iterface for staffs, enesuring all information shown is accurate and consistent.  
Built with the cobination of Customtkinter and Tkinter and MySQL for reliable backend data storage and management.

# Features
- Reservation page
  - Create Reservation - this feature allow user to create reservation
  - Update Reservation - this one allows the user to update a existing reservation and change its status where if the customer arrived or cancelled the reservation+
  - Assign  Table - this function allows the user to assign a table where customer will be seated.
- Menu page
  - Add - this one for adding meal for menu.
  - Delete - this is where meal will be delete but no permannently deleted in the database.
  - Update status and price - Update wheter the status of the specific meal is available or not and also if the price changes
- Customer page
  - Create - this where user creates customer which is walikn.
  - Add Order - this where thee creation of the order.
  - Update Customer Details 
- Order page
  -  View All current order - this is where the user can see all the current customer has order.
  -  View specific  order deatils - the user can see the details of the order. 
  -  Change Status - this where the user update the stauts if the food is served.
