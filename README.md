# Fine Dining Ordering System 
A Python-Based program application where  designed to manage the process of handling orders, reservations, and payments. It provides user-friendly iterface for staffs, enesuring all information shown is accurate and consistent.  
Built with the cobination of Customtkinter and Tkinter and MySQL for reliable backend data storage and management.

# Features
- Reservation page
  - Create Reservation
  - Update Reservation
  - Assign  Table 
- Menu page
  - Add
  - Delete
  - Update status and price
- Customer page
  - Create
  - Add Order
  - Update Customer Details
- Order page
  -  View All current order
  -  View specific  order deatils
  -  Change Status 
# Installation
  Notes: All information and functions are just sample so you need to replace them to match your design.
- Download All files or just copy link of the repository to clone.
- Make a MySqL database and design your own table based on your preference.
- Replace all information that is design for your use.
- After that make sure your database are connected to the system.
- You can now use the system based to your design.
